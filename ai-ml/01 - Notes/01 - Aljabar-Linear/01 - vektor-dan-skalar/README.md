# Vektor dan Skalar

Materi ini membahas konsep dasar tentang vektor dan skalar dalam aljabar linear, yang merupakan fondasi penting dalam ilmu komputer dan khususnya dalam Machine Learning.

## Isi Materi

1. **Pengertian Dasar**
   - Skalar: besaran yang hanya memiliki nilai/magnitude tanpa arah
   - Vektor: besaran yang memiliki nilai dan arah

2. **Operasi pada Vektor dan Skalar**
   - Penjumlahan dan pengurangan
   - Perkalian (skalar dengan vektor, dua vektor)
   - Dot product dan cross product

3. **Norma Vektor**
   - Norma-2 (Euclidean)
   - Norma-1
   - Norma tak hingga

4. **Aplikasi dalam Machine Learning**
   - Representasi data
   - Parameter model

5. **Implementasi Python**
   - Contoh kode untuk operasi vektor menggunakan NumPy

6. **Latihan**
   - Latihan dasar hingga lanjutan dengan jawaban
   - Implementasi fungsi Python untuk operasi vektor
